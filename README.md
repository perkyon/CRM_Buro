# CRM_Buro

Локальная разработка и быстрая проверка проекта.

Запуск:

```bash
npm install
npm run dev
```

Сборка для продакшна:

```bash
npm run build
npm run preview
```

Как запушить в GitHub:

```bash
git remote add origin https://github.com/<ваш-логин>/CRM_Buro.git
git push -u origin master
```
